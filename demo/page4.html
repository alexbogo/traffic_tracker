External Tracker Demo
This page demonstrates loading the tracker from an external file, similar to how you would deploy it via CDN in production.

Implementation

This page loads the tracker from:

<script src="http://localhost:8080/tracker.js"></script>

In production, you would use a CDN URL:

<script src="https://cdn.yourdomain.com/tracker.min.js"></script>

Features

  • Automatic initialization on page load
  • Browser fingerprinting for unique visitor identification
  • Session tracking across pages
  • Retry logic with exponential backoff
  • Configurable API endpoint
  • Debug mode available

Configuration Options

Customize the tracker before loading:

<script> window.TRACKER_CONFIG = { apiUrl: 'https://api.yoursite.com/api/track', debug: false, autoTrack: true }; </script> <script src="http://localhost:8080/tracker.js"></script>

Production Deployment

For production use:

  1. Upload tracker.min.js to your CDN
  2. Configure CORS on your API server
  3. Update the script source to your CDN URL
  4. Set appropriate cache headers
